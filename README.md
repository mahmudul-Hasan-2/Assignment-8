# QurbaniHat – Livestock Booking Platform

## 🎯 Purpose of the Project

The Qurbani Hat project is a modern, full-stack web application designed to bridge the gap between cattle sellers and buyers during the Eid-ul-Adha season. The primary goal is to provide a seamless, transparent, and secure digital marketplace where users can browse, manage, and facilitate the sale of sacrificial animals without the traditional physical hassles.

## 🚀 Live Demo

You can access the live version of the application here:  
**[Qurbani Hat Live](https://assignment-8-psi-red.vercel.app)**

---

### 🚀 Key Features

- 🔐 **Secure Auth:** Made by Better-auth and integrated Google.
- ⚡ **Next.js 15:** Utilizing the latest App Router and Server Actions.
- 🐄 **Cattle Management:** Easy listing and management for sellers.
- 📱 **Mobile First:** Fully responsive design using Tailwind & DaisyUI.
- 🔔 **Instant Feedback:** Integrated toast notifications for every action.

---

## 📦 Tech Stack & Dependencies

This project is built with a modern tech stack focusing on performance, smooth animations, and a great developer experience.

### 🛠️ Core

- **Next.js 15** - The React framework for production.
- **Better-Auth** - Secure and flexible authentication (Google Social Sign-in).

### 🎨 UI & UX

- **[react-spring](https://www.react-spring.dev/)** `🍃`  
  Used for physics-based animations to create a fluid and natural user interface.
- **[react-icons](https://react-icons.github.io/react-icons/)** `🎨`  
  A comprehensive library used to include high-quality icons throughout the app.
- **[react-toastify](https://fkhadra.github.io/react-toastify/)** `🔔`  
  Provides beautiful and customizable toast notifications for instant user feedback.
- **[react-hook-form](https://react-hook-form.com/)** `📝`  
  A performant way to handle form validation and submission with minimal re-renders.

### 💅 Styling

- **Tailwind CSS** - A utility-first CSS framework for rapid UI development.
- **DaisyUI** - A powerful component library built on top of Tailwind.

## 💻 Installation Guide

Follow these instructions to set up the project locally.

### 1. Prerequisites

Ensure you have Node.js (v25.9.0+) installed on your system.

### 2. Clone the Repositories

```
git clone https://github.com/mahmudul-Hasan-2/Assignment-8
```

### 3. Navigate to Directory

```
cd Assignment-8
```

### 4. Install Dependencies

```
npm install
```

### 5. Launch Development Server

```
npm run dev
```

---

### 6. See Preview

Open [http://localhost:3000](http://localhost:3000) in your browser to explore the Qurbani Hat platform. 🚀
